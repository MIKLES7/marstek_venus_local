# Marstek Venus Local (UDP)

Custom Home Assistant integration for **Marstek Venus** devices using the local UDP API.

## Installation (HACS)
1. HACS → Integrations → ⋮ → **Custom repositories**
2. Add this repository URL as **Integration**
3. Install **Marstek Venus Local (UDP)**
4. Restart Home Assistant
5. Settings → Devices & Services → Add Integration → **Marstek Venus Local (UDP)**

## Notes
- Domain: `marstek_venus_local`
- iot_class: `local_polling`
- Documentation reference: https://static-eu.marstekenergy.com/ems/resource/agreement/MarstekDeviceOpenApi.pdf
